TechForest cuida a tus plantas

Sistema de riego automatico, con opcion a manual, para el cuidado de plantas a nivel domestico como empresarial.

Ventaja 01

Ahorro de agua

Al saber la humedad de nuestro suelo logramos controlar el flujo de agua.

Ventaja 02

Mayor eficiencia de riego

Gracias a los datos que obtenemos logramos contruir un sistema de riego adecuado para nuestros tipos de plantas.

Ventaja 03

Aumentos de la producción

Al darle la adecuada cantidad de agua a nuestro sembradio, logramos que nuestras plantascrezcan mas rapido y mejor.

Tecnologias Usadas

Proyecto Integrador

Android

Sistema operativo móvil basado en el núcleo Linux y otros software de código abierto.

Django

Framework de desarrollo web de código abierto, escrito en Python, que respeta el patrón de diseño conocido como modelo–vista–controlador.

Arduino

Plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores.

Kotlin

Lenguaje de programación de tipado estático que corre sobre la máquina virtual de Java y que también puede ser compilado a código fuente de JavaScript.

Python

Lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código.​

C++

Lenguaje de programación que proviene de la extensión del lenguaje C para que pudiese manipular objetos.
